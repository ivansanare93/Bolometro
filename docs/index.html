¿Qué es Bolómetro?

Bolómetro es una aplicación móvil multiplataforma desarrollada en Flutter que permite a los jugadores de bolos llevar un seguimiento completo de su rendimiento. Diseñada tanto para jugadores aficionados como profesionales, te ayuda a tomar decisiones basadas en datos reales.

Disponible para Android e iOS, funciona tanto en modo sin conexión como sincronizando tus datos en la nube mediante tu cuenta de Google.

Características principales

🎯

Registro de Partidas

Marcador completo con validación en tiempo real, soporte para strikes, spares y el décimo frame.

📊

Estadísticas Detalladas

Analiza tu promedio, mejor partida, porcentaje de strikes y spares, y mucho más.

📈

Evolución Visual

Gráficos interactivos para ver tu progreso a lo largo del tiempo y detectar tendencias.

🏆

Gamificación

29 logros únicos, sistema de niveles con XP y 4 niveles de rareza: Común, Raro, Épico y Legendario.

👥

Sistema de Amigos

Busca amigos por correo o código, compara rankings y compite por el primer puesto.

☁️

Sincronización en la Nube

Inicia sesión con Google y accede a tus datos desde cualquier dispositivo de forma segura.

📴

Modo Sin Conexión

Todas tus partidas se guardan localmente con Hive; no necesitas internet para jugar.

🔔

Notificaciones Push

Recibe alertas al desbloquear logros, aceptar solicitudes de amistad y más novedades.

Tecnología utilizada

Bolómetro está construido con un stack moderno y fiable:

Tecnología Uso
Flutter / Dart Framework principal multiplataforma (Android, iOS, Web)
Firebase Authentication Autenticación segura con Google
Firebase Firestore Base de datos en la nube en tiempo real
Firebase Analytics Análisis de uso anónimo
Firebase Cloud Messaging Notificaciones push
Hive Almacenamiento local en el dispositivo

Privacidad y Seguridad

La privacidad de nuestros usuarios es una prioridad. Tus datos se protegen mediante conexiones cifradas (HTTPS/TLS), reglas de seguridad de Firestore y autenticación robusta. Nunca vendemos ni compartimos tu información con terceros con fines comerciales.
